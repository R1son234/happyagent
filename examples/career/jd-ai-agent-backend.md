# AI Agent Backend Engineer JD

We are hiring a backend engineer to build AI agent infrastructure for internal developer workflows.

Responsibilities:

- Design agent runtime loops that can plan, call tools, observe results, and stop safely.
- Build tool calling integrations for files, shell commands, MCP servers, and retrieval over project docs.
- Add traces, run history, and eval workflows so agent behavior can be debugged and improved.
- Implement policy controls for risky tools and keep local execution secure.
- Package the product as a reliable CLI with clear reports and reproducible demo flows.

Requirements:

- Strong Go backend engineering experience.
- Practical LLM API integration and structured output handling.
- Experience with RAG or documentation search.
- Ability to explain architecture, tradeoffs, and validation strategy.
- Bias toward simple, inspectable systems over opaque prompt-only wrappers.
